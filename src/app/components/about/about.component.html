Art Institute of Chicago Artwork Search and Viewing Application Documentation Page

This page will provide you with information on how to use the application and its features.

Getting Started

Just go to the main page of the application to begin using it. has a carousel and a navigation bar with fields for "home," "favorites," "about," "collections," Users can switch between pages by clicking on these fields.

Search features

You can use this search area to locate the information you're looking for or wish to see.You can look up information about artists, their names, or any other art-related terms.

Pagination

For large result sets, pagination will be implemented to ensure smooth user experience.

Collections Page

This web page features a variety of artworks arranged in a grid style for convenient viewing. Every piece of art has an image, a title, and the name of the artist.

Artwork features

Share features

You can use social media apps like Facebook, Twitter, LinkedIn, and others to share the details of an artwork with your friends by clicking on the share button found on that specific art card.

More Details

To get more details about artwork, here is the detailed page, which consists of the artist, medium, dimensions, date, credit line, category, description, publication history, and exhibition history of the artwork that you want to explore.

Authenciation

Register page

In order to create a new user account in my application, the registration process requires a unique username and email address and a strong password.

Login page

In order to access their account in my application, the login process requires a registered username and password.

Favorites feature

Favorites page

Clicking on the artwork allows authorized users to favorite it.

Favorites Page

All of the logged-in user's favorite artwork is available on this favorite page.
